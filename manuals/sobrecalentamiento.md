# Sobrecalentamiento del servidor

## Síntomas
- Temperatura del chasis por encima de 75 °C.
- A menudo acompañado de un uso de CPU elevado (sobrecarga térmica).

## Causa probable
El sistema de refrigeración no consigue disipar el calor generado. Las causas habituales son
ventiladores obstruidos o averiados, filtros de aire sucios, temperatura ambiente del CPD
demasiado alta, o disipadores mal asentados. El calor sostenido acelera el desgaste del
hardware y puede provocar apagados de protección o caídas.

## Acciones recomendadas
1. Verificar que todos los ventiladores giran y no están obstruidos.
2. Revisar la temperatura ambiente del rack / CPD (objetivo por debajo de 24 °C).
3. Limpiar filtros de aire y comprobar el flujo de aire frontal y trasero.
4. Si la carga de CPU es alta, redistribuir procesos a otros nodos para bajar la temperatura.
5. Si persiste, programar una parada para revisar la pasta térmica y los disipadores.
